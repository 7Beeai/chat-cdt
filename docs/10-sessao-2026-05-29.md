## Como verificar uma RPC SECURITY DEFINER sem browser (receita)

`auth.uid()` é null no MCP, então a função filtra tudo e parece vazia. Para testar de verdade, **impersone um usuário** numa transação:

```sql
select set_config('request.jwt.claims',
  json_build_object('sub','<auth_user_id>','role','authenticated')::text, true);
select chat_report_overview(now() - interval '30 days', now(), null);
```

(Ian = `c89111c5-2bfe-4254-b2cf-97619f6898b5`, tem as 13 unidades.) Isso valida scope, fuso e coalesce de verdade. **`next build` verde NÃO testa a camada de dados** — sempre faça isto + a validação visual.
